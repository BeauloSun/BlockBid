## File structures:

- Contracts: Storing smart contracts
- node_modules: Currently has nothing to do with the project
- scripts: Store js files used to deploy our smart contracts
- test: Store test to test the smart contract using hardhat
- hardhat.config.js: Configuration for the network
- package.json: All the information for the current project

### Set-up steps:

1. Install VSCode extensions: solidity(language support) and Solidity(by hardhat)
2. Install Nodejs with automatic installer
3. Install hardhat: npm install --save-dev hardhat
4. Run hardhat using: npx hardhat init ; copy the additional package installation command from prompt

### Windows specifics:

1. Install VSCode extensions: solidity(language support) and Solidity(by hardhat)
2. Use powershell to install Ubuntu terminal:
   wsl --install (if not installed) / wsl --update (if not version 2)
   wsl --list --online (check for available systems)
   wsl --install -d Ubuntu-20.04
   **_(To change versions, use the command: wsl --set-version <distro name> 2 replacing <distro name> with the name of the Linux distribution that you want to update. For example, wsl --set-version Ubuntu-20.04 2 will set your Ubuntu 20.04 distribution to use WSL 2.)_**
3. Install nodeJS from:
   - https://nodejs.org/en/download/current
4. Then, in the Ubuntu terminal, follow this link:
   - https://www.educative.io/answers/how-to-install-solidity-on-windows
5. Install npm[node package manager] from:
   - https://www.npmjs.com/
